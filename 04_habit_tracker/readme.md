# habit-tracker
An XP/RPG-style habit designed to gamify daily habits using XP, streaks, and progression system. 

Current Features:
- Track daily habits
- Earn XP for completing tasks
- Persistent save system for XP and streaks
- Basic  XP and streak tracking 
- Simple terminal interface
- Menu-based navigation

Current Tech Stack: 
- Python 3
- JSON or text file for saving data
- Standard python modules
- Runs entirely in the terminal 

Planned Updates:
- Break the program down into functions 
- Add a daily reset feature that detects when the date changes and automatically update the current streak (currently only updates when habits are completed)
- Add error handling + input validation
- Expand the XP system (levels, rewards, multipliers)
- Add more habit customization options
